# IMEStateViewer

IMEStateViewerは、現在のIMEの入力状態（日本語入力か英語入力か）に応じて、画面の縁にグラデーションを表示するWindows向けのオーバーレイツールです。
現在の入力状態を視覚的に把握しやすくすることで、入力ミスを防ぐことができます。

## 特徴

- **リアルタイムな状態検知**: アクティブなウィンドウのIME状態をリアルタイムで取得し、日本語入力（ひらがな等）か英語入力（半角英数字）かを判定します。
- **画面の縁のグラデーション**: 状態に応じて画面の縁にグラデーションを描画します。
- **マルチディスプレイ対応**: 現在アクティブなウィンドウが存在するディスプレイに自動的に追従して表示されます。
- **カスタマイズ可能な色設定**: コントロールウィンドウから、日本語入力時と英語入力時の色（透明度含む）を自由に変更できます。
- **入力の邪魔にならない**: オーバーレイは入力透過（マウスクリックなどを下のウィンドウに透過）に設定されているため、作業の邪魔になりません。

## 動作環境

- Windows OS (Windows APIを使用しているため、Windows専用です)
- Python 3.x
- PySide6

## インストール

1. Pythonがインストールされていることを確認します。
2. 必要なライブラリである `PySide6` をインストールします。

```bash
pip install PySide6
```

## 使い方

以下のコマンドでアプリケーションを起動します。

```bash
python code.py
```

### コントロールウィンドウの操作

アプリケーションを起動すると、画面の縁にグラデーションが表示されるとともに、小さなコントロールウィンドウが表示されます。

- **日本語入力時の色**: ボタンをクリックするとカラーピッカーが開き、日本語入力時の色と透明度を変更できます。
- **英語入力時の色**: ボタンをクリックするとカラーピッカーが開き、英語入力時の色と透明度を変更できます。
- **アプリケーションを終了**: このボタンをクリックすると、アプリケーション全体が終了します。

## 実行ファイル（.exe）の作成方法

Python環境がないPCでも実行できるように、PyInstallerを使って単一の実行ファイル（.exe）に変換することができます。

1. PyInstallerをインストールします。
```bash
pip install pyinstaller
```

2. 以下のコマンドを実行してビルドします。`--noconsole` オプションで実行時にコマンドプロンプトを非表示にし、`--icon` オプションで実行ファイルのアイコンを設定します。また、プログラム内からアイコンを読み込めるように `--add-data` オプションでアイコンファイルを同梱します。
```bash
pyinstaller --noconsole --onefile --icon=IME.ico --add-data "IME.ico;." code.py
```

3. ビルドが完了すると、`dist` フォルダの中に `code.exe` が生成されます。このファイルを任意の場所に配置して実行してください。
